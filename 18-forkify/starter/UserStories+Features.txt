As a user, I want to search for recipes, so that I can find neww ideas for meals

As a user, I want to be able to update the number of servings, so that I can cook a meal for a different number of people each time

As a user, I want to bookmark recipes, so that I can review them later

As a user, I wanted to be able to create my own recipes, so that I have them all organised in the same app

As a user, I want to be able to see my bookmarks and own recipes when I leave the app and return later, so that I can close the app safely after cooking



Features
search for recipes -> 
    search functionality
    display results with pagination
    display recipe with cooking time, servings and ingredients


update the number of servings ->
    functionality to change all ingredient amounts dependant on serving settings

bookmark recipes ->
    bookmarking functionality
    display list of all bookarked recipes

create my own recipes
    User can upload own recipes
    User recipes will automatically be bookmarked
    User can only see their own recipes, not recipes from other users

see bookmarks and own recipes when leaving the app and returning later
    Store bookmark data in the browser using local storage
    On page load, read saved bookmarks from local storage and display
